เลขยกกำลังแบบไว (ม.ต้น)

ฐาน 1–12 · ฐาน 6, 7, 8, 9, 11 และ 12 ใช้ได้แค่กำลัง 1 หรือ 2 · รวมกำลังศูนย์ และโจทย์ลบซ้อนแบบ −(−a)n / −(−an) เป็นบางข้อ — เลือกคำตอบจาก 4 ช่อง (แนะนำ 3 หรือ 5 วินาที)

กดปุ่มด้านล่างเพื่อเปิดดู สมบัติ · สูตร · ตารางอ้างอิง แบบย่อ — พร้อมแล้วค่อยปิดเพื่อโฟกัสเกม

สมบัติและสูตร

สรุปเบื้องต้นสำหรับเลขยกกำลัง

  • a0 = 1 เมื่อ a ≠ 0 · a1 = a
  • 1n = 1 ทุก n · ฐาน 0 ยกกำลังบวกได้ 0 (เช่น 03 = 0)
  • (−a)n ยกกำลัง «ทั้งฐานในวงเล็บ» — n เป็นเลขคู่ → ผลเป็นบวก · n เป็นเลขคี่ → ผลเป็นลบ (เมื่อ a > 0)
  • −an = −(an) ลบอยู่หน้านิพจน์ ไม่ใช่ฐานลบ — ต่างจาก (−a)n
  • ลบซ้อน: −(−a)n คิด (−a)n ก่อน แล้วใส่ลบหน้า · −(−an) ในวงเล็บเป็น −(an) ก่อน แล้วลบข้างนอกชนกลาง
